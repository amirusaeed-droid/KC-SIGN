User Guide / Tutorial

1. Upload PDF: Select the PDF document from your phone or computer.

2. Add Signature: Draw, type, or upload your signature image. Then press Use Signature.

3. Add Seal: Upload a seal/stamp image or use the Kondey Council logo as the seal.

4. Move Fields: Drag signature, seal, name, date, or text to the correct place. Use the corner dot to resize.

5. Use More Than Once: Select a field and press Duplicate Selected to place the same signature or seal again.

6. Multi-page PDF: Scroll down to view all pages. Use zoom + / − to zoom only the PDF pages.

7. Download: Press Download Signed PDF when finished.
